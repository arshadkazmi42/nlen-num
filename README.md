Generate random number of n length.

## Install

```
npm i nlen-num
```

## Usage

```javascript
const nLenNum = require("nlen-num");

console.log(nLenNum(3));
console.log(nLenNum(4));
console.log(nLenNum());
console.log(nLenNum(0));

// OUTPUT
// 132
// 4323
// 562
// 456
```

> Note: Default length is 3, if there is no length input passed
